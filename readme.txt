  概要
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ローカル動作のネットワーク構成図エディタです。
インフラ構成のレビュー資料・教育資料の作成を主な用途として設計しています。

「描けること」より「見て理解しやすいこと」を優先した設計です。
通信経路・セグメント所属・役割を視覚的に分離することで、
第三者が見ても意図が伝わる構成図を手早く作れます。

  動作環境・技術要件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  OS          : Windows 10 / 11
  Node.js     : 18 以上
  Rust        : 1.70 以上  (rustup 経由でインストール推奨)
  フレームワーク: Tauri v2 + React 18 + TypeScript
  ビルドツール  : Vite 5
  状態管理      : useReducer (React 組み込み)
  永続化        : @tauri-apps/plugin-store (ローカルJSON)
  外部UIライブラリ: なし (カスタムSVGキャンバス実装)

  主要機能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【ノード (NetworkNode)】
  種別 : server / loadbalancer / database / monitor /
         user / firewall / router
  属性 : 表示名, ホスト名, IPアドレス, 役割, 備考
  タグ : セグメントタグ (所属ネットワーク, 複数可),
         ユーザータグ (任意)

  追加方法:
    - 左パネルのボタンをクリック → キャンバス中央に追加
    - キャンバスをダブルクリック → クリック位置に追加
      (追加される種別は左パネルで最後に選択したもの)

  移動: ドラッグ
  編集: クリック → 右パネルで各属性を編集

【接続線 (NetworkEdge)】
  方向  : 一方向 (→) / 双方向 (↔)
  通信種別 (線スタイルで区別):
    業務通信 … 青・実線
    管理通信 … 緑・破線
    監視通信 … 紫・点線
    カスタム … グレー・実線
  プロトコル: SSH / HTTP / HTTPS / PostgreSQL /
              MySQL / SNMP / ICMP / DNS / NTP / SMTP
              (複数付与可能)
  ラベル  : 自由テキスト

  作成方法:
    1. ツールバーの [Select ON] をクリック
    2. 接続元ノードをクリック (番号バッジが表示される)
    3. 接続先ノードをクリック
    4. [Link →] または [Link ↔] をクリック

  編集: 接続線をクリック → 右パネルで種別・プロトコル等を編集
  削除: 右パネルの [この接続線を削除]

【ネットワーク領域 (NetworkArea)】
  セグメントを背景矩形で表現します。
  デフォルト名: MGMT / DMZ / APP / DB / MON
  カスタム名も追加可能。

  追加: 左パネルのボタンをクリック
  移動: 領域内をドラッグ (中に含まれるノードも連動して移動)
  リサイズ: 右下ハンドルをドラッグ
  名前変更: 領域名をダブルクリック
  削除: 領域名右の [×] ボタン

【キャンバス操作】
  ズーム      : マウスホイール
  パン        : 背景をドラッグ、または中ボタンドラッグ
  グリッドスナップ: ツールバー [Grid ON/OFF]
  ビューリセット  : ツールバー [⌖ Home]

【ショートカット】
  Ctrl+Z          : Undo
  Ctrl+Y          : Redo
  Ctrl+Shift+Z    : Redo (代替)
  ダブルクリック   : ノード追加 (キャンバス背景上)

【保存・エクスポート】
  自動保存: 操作のたびにローカルストレージへ自動保存
  Export JSON: 図全体をJSONファイルとして書き出し
  Import JSON: JSONファイルを読み込み (現在の図は上書き)
